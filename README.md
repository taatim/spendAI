### Installation
1.  Clone the repo:
    ```bash
    git clone https://github.com/taatim/spendwise_mobile.git
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the app:
    ```bash
    flutter run
    ```
